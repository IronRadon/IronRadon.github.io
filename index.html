Project Highlights

  • GoodBoards(2014)

Lightweight boardgame clone of GoodReads where users can review and rate board games. Data that is missing from the database is dynamically mined and added to the database while views are rendered to create a smooth user experience. Technologies: Ruby, Rails, Javascript, Backbone.js, XML, HTML, CSS

  • ActiveRecod Clone(2013)

Lightweight clone of ActiveRecord's ORM features. Mass assignment, querying, association methods. Extensive use of Ruby's metaprogramming features. Technologies: Ruby, Metaprogramming
